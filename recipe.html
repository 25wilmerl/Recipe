-Main folder: index.html, folders called pages, img -Index.html is going to list the recipes in your cookbook. These recipes should be listed as an unordered list. -in Pages, you'll have at least 5 separate html pages, one per recipe. (e.g. tacos.html) -Each page should have the name of the recipe in an

tag -each page should have a description of the food, a header called ingredients, an unordered list of ingredients, a header called "steps", and an ordered list of the steps you use to make the dish. -Each page should have an image of the dish Index page should be linked to your recipe pages, and your recipe pages should have a link at the bottom to go back to the main page (index)
